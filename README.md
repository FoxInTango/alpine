# alpine
Auto and Remote System Managing for Linux.

# USAGE:
###  git clone https://github.com/FoxInTango/alpine.git
###  make subinstall
###  make submodule OPENSSL= ABS/PATH/TO/OPENSSL/ROOT [include & lib]
###  make 
###  make install

# UPDATE:
###  make update

# For Developing
###  make devinstall 
###  make update
###  make publish

###  make && make install
