# alpine
Auto and Remote System Managing for Linux.

# usage:
###  git clone https://github.com/FoxInTango/alpine.git
###  make subinstall
###  make subupdate
###  make submodule

# For Developing
###  git submodule init
###  git submodule update --remote --recursive

###  make subupdate
###  make submodule

###  make OPENSSL=../thirds/openssl

###  make install
