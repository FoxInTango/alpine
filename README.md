# alpine
Auto and Remote System Managing for Linux.

# usage:
###  git clone https://github.com/FoxInTango/alpine.git
###  make subinstall
###  make subupdate
###  make submodule OPENSSL=../thirds/openssl
###  make

# For Developing
###  make devinstall 
###  make subupdate
###  make devpublish

###  make && make install
